Installing GIMMIE

Add the X Bookmark Exporter to your browser in about two minutes. No coding, no command line.

ⓘ
Why these steps? GIMMIE is being reviewed for the Chrome Web Store. Until it is approved, you add it manually with the steps below. Once it is on the store, it will be a single click. These steps work in Chrome, Edge, Brave, and other Chrome-based browsers.
1

Download GIMMIE

On the GIMMIE GitHub page, click the green Code button, then Download ZIP. (If someone shared a ZIP with you directly, just use that.)

2

Unzip it, and keep the folder

Double-click the downloaded ZIP to unzip it. You will get a folder with files inside it, including one called manifest.json.

Put this folder somewhere permanent (like your Documents), because GIMMIE runs from it. If you delete or move the folder later, the extension stops working.

3

Open your browser's Extensions page

In the address bar, type chrome://extensions and press Enter. On Edge, use edge://extensions.

You can also reach it from the menu: ⋮ → Extensions → Manage Extensions.

4

Turn on Developer mode

Find the Developer mode switch in the top-right corner of that page and turn it on. A new row of buttons will appear.

5

Click "Load unpacked" and pick the folder

Click Load unpacked (top-left). In the file picker, select the folder you unzipped in step 2 (the one containing manifest.json) and confirm.

GIMMIE now appears in your list. That's it, it's installed.

6

Pin it so it's easy to find

Click the puzzle-piece icon 🧩 near the address bar, then click the pin next to GIMMIE. Its icon will stay in your toolbar.

7

Use it

Go to x.com/i/bookmarks, click the GIMMIE icon, and choose Grab everything or Browse and pick. Then export as Markdown, PDF, JSON, CSV, or ZIP.

✓
You're in control. GIMMIE runs entirely on your computer and only reads your own bookmarks. Nothing is uploaded anywhere. See the privacy policy in the project for details.

Troubleshooting

"Load unpacked" is greyed out or missing

Developer mode isn't on yet. Flip the switch in the top-right of the Extensions page (step 4), then try again.

It asked me to pick a file, but nothing happened

You need to select the whole folder, not a single file inside it. Pick the folder that directly contains manifest.json.

The extension disappeared or stopped working

The folder was probably moved, renamed, or deleted, or it was still inside the ZIP. Unzip it, put the folder in a permanent spot, and load it again from there.

GIMMIE opens but finds nothing on my bookmarks

Make sure you are signed in to X and actually on your bookmarks page (x.com/i/bookmarks), then scan again. X occasionally changes its site, which can require a GIMMIE update, if it keeps happening, please report it on the project page.

How do I update to a newer version?

Download the new version, unzip it over the same folder (replacing the files), then click the refresh icon on GIMMIE's card in the Extensions page.

Is it safe to keep Developer mode on?

Yes. Developer mode only lets you install extensions from a folder like this one. You can leave it on, or turn it off after installing, GIMMIE keeps working either way.
